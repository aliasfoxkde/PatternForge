## Quick Summary

This document provides a comprehensive feature inventory of StitchFiddle (the market leader in browser-based multi-craft pattern design) and its primary competitors. Data was gathered by directly fetching and parsing the live websites.

## 1. STITCHFIDDLE.COM -- Comprehensive Feature Inventory

### 1.1 Company Overview

- **Founded:** March 2015 (registered company December 2015)
- **Platform:** 100% browser-based (no download/install required)
- **Tagline:** "Online knitting and cross stitch chart pattern editor software"
- **Vision:** "Helping crafters spend less time on designing, to have more time for crafting"
- **Mission:** "Make designing easy, accessible and intuitive to every crafter"
- **Pricing model:** Freemium (free tier stays free forever; premium is one-time payments, no auto-renewal)
- **Social:** Facebook page, Facebook user group, Instagram, Ravelry group, Twitter, Pinterest, LinkedIn

### 1.17 Technical Notes

- React-based SPA (single page application)

- CSS versioned by build timestamp
- JS chunked loading (main + named chunks)

- Block-all-mixed-content policy
- Server-side state serialization (base64-encoded JSON in HTML)
- Internationalization support (en, nl languages)
- SEO optimized (OG tags, Twitter cards, robots meta)
- Google translation disabled (notranslate class)

### 1.18 Known Limitations (from competitor analysis)

- Image-to-pattern conversion produces more "confetti" (isolated stitches) than competitors
- No FLOW score or pattern quality analysis
- No confetti cleanup tools
- No French knots support
- No blended threads support
- Limited selection tools (no lasso, magic wand)
- No Pattern Keeper-compatible PDF export
- Limited pattern file import (only .OXS)
- Free tier severely limits drawing tools (no fill, mirror, copy/paste)
- Written instructions only for knitting/crochet (not cross stitch)
- No GPU-accelerated rendering
- No standalone desktop app
- No fabric calculator, time calculator, or thread calculator
- No realistic stitch preview/mockup
- No pattern size estimations

## 2. STITCHMATE.APP -- Feature Inventory

### 2.1 Company Overview

- **Creator:** Dmitrii (solo developer)
- **Platform:** Browser-based (any modern browser) + desktop app coming 2026
- **Focus:** Cross stitch only (specialized)
- **Pricing model:** Freemium with one-time payments (no subscriptions)
- **Claim to fame:** 79% less confetti than StitchFiddle with default settings

### 2.2 Supported Craft Types

- **Cross stitch ONLY** (deeply specialized)

### 2.3 Key Differentiating Features

**FLOW Score (0-100):**
- Fragmentation Score (confetti density)
- Locality Score (color clustering quality)
- Optimization Score (palette problems like similar colors)
- Workability Score (practical difficulty)
- Actionable Issues List (specific problems to fix)

**Confetti Reduction:**
- Global confetti reduction (one-click cleanup)
- Brush-based cleanup (local area cleanup)
- Before/after preview
- Adjustable threshold

**Photo Conversion:**
- Spatial-aware quantization (not pixel-independent)
- Perceptual color matching (CIEDE2000 in OKLCH color space)
- Real-time preview as you adjust settings
- Pixel Art mode (each pixel = one stitch)

### 2.4 Drawing Tools

- Pencil tool (single stitch placement)
- Brush tool (adjustable 1-20 cells)
- Fill tool (flood fill)
- Eraser tool (adjustable brush)
- Color picker (click to grab color)
- Text tool (multiple cross-stitch fonts, live preview)
- Image overlay tool (trace over reference photo)
- Multi-layer editing (lock, hide, merge)
- Rectangle, ellipse, line shape tools

### 2.5 Selection & Transform

- Rectangle selection
- Lasso selection (freehand)
- Magic wand (connected color region)
- Move & copy
- Flip & mirror
- Rotate (90-degree steps)

### 2.6 Stitch Types

- Full cross stitch
- Half stitch
- Petite stitch (4 tiny stitches per grid square)
- Three-quarter stitch
- Backstitch (with B&W-friendly legend)
- French knots (fully supported in editor, PDF, and imports)
- Blended threads (2-4 strands, across all 50+ brands)

### 2.7 Thread/Color Management

- 50+ thread brand catalogs (7,000+ colors)
- DMC, Anchor, Cosmo, Madeira, Classic Colorworks, CXC, Gentle Art, J&P Coats, Kreinik, Presencia, Sullivans, WDW, hand-dyed, silks, metallics
- Thread brand remapping (1-click full pattern conversion)
- Color replacement
- Color adjustment tools (brightness, contrast, hue shift, desaturation)
- Stitch count per color
- Perceptual color science matching (not lookup tables)

### 2.8 Export Options

- PDF export (multi-page, symbols, gridlines, page navigation)
- Pattern Keeper compatible (all paid plans)
- Thread shopping list (codes, names, stitch counts, skein estimates)
- PNG image export (free)
- Cover page (custom image for Pro)
- Customizable layout (A4/Letter, margins, color/B&W, cell size, tile overlap, legend order)
- Color legend customization (notes per color)
- SVG export (paid plans)
- Realistic mockup previews (frame/hoop)

### 2.9 Import

- Photo conversion (PNG, JPG, GIF, WebP)
- Pixel art mode
- Blank canvas
- Pattern file import: .PAT (PCStitch), .OXS (WinStitch/MacStitch), .XSD (XSPro) -- drag & drop
- Image overlay for tracing

### 2.10 Free Tools (separate from editor)

## 3. OTHER COMPETITORS (from training data + web research)

### 3.1 PCStitch

- **Platform:** Windows only
- **Price:** ~$40-50 (one-time)
- **Strengths:** Mature software, large built-in library of fonts/motifs, custom stitch creation
- **Weaknesses:** Windows-only, dated interface, no major update since Windows 7 era
- **Key Features:** Photo import, custom stitches, DMC/Anchor support, print layouts
- **Pattern Keeper:** Manual setup required

### 3.2 WinStitch / MacStitch (Ursa Software)

- **Platform:** Windows (WinStitch) / Mac (MacStitch)
- **Price:** ~$30-50 (one-time)
- **Strengths:** Cross-platform, good value, frequent updates, layers support
- **Weaknesses:** Less polished UI, fewer advanced automation features
- **Key Features:** Photo-to-chart, thread palette, backstitch, French knots, layers, PDF/image export
- **Pattern Keeper:** Premium tier
- **Import:** .OXS format

### 3.3 FlossCross

- **Platform:** Browser-based
- **Price:** Free
- **Strengths:** Completely free, simple interface
- **Weaknesses:** Limited features, only DMC and Anchor (2 brands), max 300x300, basic confetti cleanup only
- **Import:** .OXS format

### 3.4 Pattern Maker (HobbyWare)

- **Platform:** Windows only
- **Price:** ~$30-60
- **Strengths:** Professional-level, advanced photo conversion with blending
- **Weaknesses:** Older software, infrequent updates, higher learning curve

### 3.5 Cross Stitch Professional

- **Platform:** Windows
- **Price:** ~$35-80
- **Features:** Photo-to-chart, multiple stitch types

### 3.6 KG-Chart / KCross

- **Platform:** Linux / Windows
- **Price:** Free (open source)

### 3.7 Pic2Pat

- **Platform:** Browser-based

## 5. KEY GAPS & OPPORTUNITIES FOR PATTERNFORGE

Based on this competitive analysis, here are potential differentiators:

### Things NO competitor does well:

1. **AI-powered pattern generation** -- none use AI/LLMs for design assistance
2. **Natural language pattern creation** -- "make me a 50x50 rose pattern"
3. **Intelligent color palette suggestions** -- AI-recommended palettes based on image analysis
4. **Pattern difficulty scoring** -- only Stitchmate has FLOW, but it's algorithmic not AI
5. **Cross-craft pattern translation** -- convert a cross stitch pattern to knitting or vice versa
6. **Smart confetti reduction with AI** -- understanding image semantics to reduce noise intelligently
7. **Real-time collaboration** -- StitchFiddle has basic sharing, no real-time co-editing
8. **Pattern marketplace / community features** -- no built-in marketplace for selling patterns
9. **AR/VR preview** -- see your pattern on actual fabric/finished product
10. **Voice-controlled design** -- accessibility feature no competitor offers
11. **Pattern versioning** -- git-like version history for pattern iterations
12. **AI stitch counting / time estimation** -- Stitchmate has calculators but not AI-powered

### Things competitors do that should be matched:

1. **Multi-craft support** (StitchFiddle leads)
2. **Progress tracking** (StitchFiddle only)
3. **Written instructions generation** (StitchFiddle only)
4. **Large pattern support** (1000x1000)
5. **50+ thread brand databases**
6. **Pattern Keeper compatible export**
7. **Browser-based + offline desktop app**
8. **Collaboration features**
9. **Comprehensive export formats** (PDF, SVG, Word, Excel, PNG)
10. **Image-to-pattern with confetti reduction**
